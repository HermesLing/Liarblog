git 学习

git 常用指令

学习如何用git管理自己的项目，学习git的一些常用指令。

  • git init：创建库
  • git status：查看仓库的当前状态
  • **git diff <filename>**：对比文件修改后的差别

git diff readme.txt

  • **git add <filename>**：将文件添加到仓库

git add readme.txt
git add –all添加所有工作区文件

  • **git commit -m “<note>”**：将文件提交到仓库

git commit -m “wrote a readme file”

  • git log：查看历史仓库版本

git log –pretty=oneline
head表示当前版本

  • git reset：回退版本

git reset –hard HEAD^
git reset –hard HEAD~100
git reset –hard 96d60

  • git reflog：记录每一次命令
  • git checkout：撤销修改

git checkout – readme.txt

  • rm <filename>

rm test.txt

  • **git rm <filename>**：从版本库中删除

git rm test.txt

一些经验

更改git默认初始化位置（默认为C盘）

  • 右键Git Bash图标，选属性
  • 起始位置改为你自己的仓库位置
  • 目标后面的cd-to-Home 要删掉，否则还是会在C盘

warning: LF will be replaced by CRLF in <filename>

输入命令 ：

  • git config core.autocrlf false (仅对当前git仓库有效）
  • git config –global core.autocrlf false (全局有效，不设置推荐全局）
